Transformation¶

We cleaned the dataset from invalid records and we added more informative columns.

Columns gold, silver, bronze:¶

We noticed that the number of medals won of each type (gold, silver, bronze) is mostly 0 or 1 (as shown in the dataframes above).

We extracted the main and important information from those columns, by adding columns of:

  1. total medals won
  2. if the athlete won any medals
  3. if the athlete won any gold medals

(Even though column 2 indicates if the athlete won any medals, we decided to add column 3 because it represents an even greater achievement)

Therefore, we decided to drop the columns: gold, silver, bronze.

Columns sex, dob:¶

We transformed these columns into the columns 'male' and 'age' respectively. Under these columns, the values are more informative and easy to use.

Columns id, name:¶

We also decided to drop the columns: id, name, because they don't contribute any valuable information.

# functions to create the new columns
def won_medal(gold, silver, bronze):
  if int(gold) or int(silver) or int(bronze):
    return 1
  return 0

def won_gold(gold):
  if int(gold):
    return 1
  return 0

def total_medals(gold, silver, bronze):
  return int(gold) + int(silver) + int(bronze)

def get_age(dob):

  olympic_date = [8, 5, 2016]
  dob = [int(x) for x in dob.split('/')]

  if dob[2] <= 5:
    dob[2] += 2000

  else:
    dob[2] += 1900

  return olympic_date[2] - dob[2] - ((olympic_date[0], olympic_date[1]) < (dob[0], dob[1]))

def get_is_male(sex):
  if sex == 'male':
    return 1
  return 0

def convert_height(height):
  return int(100 * float(height))

def convert_weight(weight):
  return int(weight)

# turn functions into udfs
won_medal_udf = f.udf(won_medal, IntegerType())
won_gold_udf = f.udf(won_gold, IntegerType())
total_medals_udf = f.udf(total_medals, IntegerType())
get_age_udf = f.udf(get_age, IntegerType())
get_is_male_udf = f.udf(get_is_male, IntegerType())
convert_height_udf = f.udf(convert_height, IntegerType())
convert_weight_udf = f.udf(convert_weight, IntegerType())

def count_missing_values(data, col_name):
  """Gets a dataframe 'data' and string 'col_name' and prints the number
  of missing values the column 'col_name' has."""
  count_original = data.count()
  count_valid = data.filter(col(col_name).cast('string') != 'null').count()
  null_num = count_original - count_valid
  print(f"For column '{col_name}',{' '*(12-len(col_name))}there are {null_num} null values.")

Columns Description:¶

  • nationality - the nationality of the athlete (categorical variable, represented by the country's code).
  • sport - the sport field of the athlete ((categorical variable, represented by the sport's name).
  • age - the age of the athlete (numerical variable, represented by int).
  • male - the sex of the athlete (binary variable, 0 represents female and 1 represents male).
  • height - the height of the athlete in meters (numerical variable, represented by float).

  • won_any - indicates if an athlete won any medals (binary variable).
  • won_gold - indicates if an athlete won any gold medals (binary variable).
  • total_medals - the number of medals an athlete won (numerical variable, represented by int).

def statistical_desc(data, filterd_data, col_name, bin_num, units=None, xticks=None):
  """
  Gets a dataframe, the filtered dataframe, a column name, and shows
  statistics about the data, as well as plot a histogram of the filtered data
  with 'bin_num' bins, and units for the x values if necessary.
  """
  print("Statistical Description")
  data.describe(col_name).show()

  col_values = [x[0] for x in filterd_data.select(col_name).collect()]

  plt.title(f"Distribution of Different {col_name.title()} Values")
  plt.ylabel("athletes count")

  if units is not None:
    col_name += f" ({units})"

  if xticks is not None:
    plt.xticks(xticks)

  plt.xlabel(col_name)

  plt.hist(col_values, bins=bin_num, edgecolor='black', linewidth=1, color="cornflowerblue")

  plt.show()

2. Initial Analysis of Our Data Set¶

Age Column:¶

As shown below under the statistical description, the min age is 13 and the max age is 62.

Although 13 seems quit young, upon re-searching online we found out that according to the offical Olympic website, there's no age limit for taking part in the Olympic Games.

Regarding age 62, we found that the top 5 oldest atheletes participates in equestrian (horse riding), which is a less physical deminding sport.

Therefore, we can conclude that all the values under this column are valid.

According to the "minimal ink-paper ratio" principal, we chose to remove all values greater than 45 while displaying the graph (the distribution is clear and displaying the entire long tail is unnecessary).

def create_pie_chart_binary_feature(data, col_name, title):
  """
  Gets a dataframe, column and title, and plots a pie chart displaying
  the amount of each unique value in the column of the data.
  """
  lst = data.groupBy(col_name).count().collect()
  labels = [x[0] for x in lst]
  count = [x[1] for x in lst]

  plt.pie(count, labels=labels, autopct='%1.2f%%', startangle=90,
          wedgeprops={'edgecolor':'black'})
  plt.title(title)

  plt.show()

def create_horizontal_bar_plot_categorical_feature(data, col_name, figsize_tuple):

  values = [x[0] for x in lst]
  count = [x[1] for x in lst]

  # for changing the col_name noun into plural
  engine = inflect.engine()

  plt.figure(figsize=figsize_tuple)

3. Research Questions¶

First question type:

  • Among the athletes who won a medal, does increase in the athlete's height cause the number of medals won to increase?
  • Among the athletes who won a medal, does increase in the athlete's weight cause the number of medals won to decrease?

Second question type:

  • Does increase in the athlete's weight lead to an increase in the probabilty to win a medal?
  • Among the athletes who participate in wrestling, does decrease in the athlete's weight lead to a decrease in the probabilty to win a medal?
  • Among the athletes who participate in athletics, does decrease in the athlete's height lead to an increase in the probabilty to win a medal?

Third question type:

  • Does the average height of the athletes change between the athletes who won a medal and the ones who didn't?
  • Does the average weight of the athletes change between the athletes who won a medal and the ones who didn't?
  • Does the average height of the athletes change between the different sport fields?
